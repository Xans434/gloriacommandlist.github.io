Gloria

A self-written bot for the Discord server of fucking scum, created out of boredom and curiosity >:3

For Everyone

/checkid
Retrieves and displays a paginated list of all server categories, text channels, and voice channels alongside their respective IDs.
Usage: /checkid
/cooldown
Displays the remaining cooldown timers for a user's customization commands. If no user is specified, it checks the command executor.

Parameters: user (User, optional) - The target user to check.
Usage: /cooldown OR /cooldown user:@username
/customize avatar
Updates the server's avatar. A cooldown applies to non-owner users. Requires either an image URL or an attached image file.

Parameters: text (string, optional) - Direct image URL, or image (attachment, optional) - Uploaded image file.
Usage: /customize avatar text:https://... OR image:[file]
/customize channel
Renames a specified text channel. A cooldown applies to non-owner users.

Parameters: channel (TextChannel) - The target channel, text (string) - The new channel name.
Usage: /customize channel channel:#general text:new-name
/customize name
Updates the server's name. A cooldown applies to non-owner users to prevent spam.

Parameters: text (string) - The new server name.
Usage: /customize name text:My Awesome Server
/help
Provides a link to the official Gloria bot website for detailed documentation and command references.
Usage: /help
/serverinfo
Displays comprehensive server information, including name, ID, member count, creation date, owner, and the current status of Anarchy mode.
Usage: /serverinfo

For the Owner

/anarchy
Toggles "Anarchy" mode. When enabled, customization command cooldowns are bypassed for all users.

Parameters: status (boolean) - True to enable, False to disable.
Usage: /anarchy status:True
/clear
Purges a specified number of recent messages in the current channel. Requires the "Manage Messages" permission.

Parameters: number (integer, 1-100) - The amount of messages to delete.
Usage: /clear number:50
/database
Generates and sends a backup of the cooldown database file directly to the bot owner via Direct Message.
Usage: /database
/reset_cooldown
Instantly clears all customization command cooldowns for the specified user.

Parameters: user (User) - The target user whose cooldowns will be reset.
Usage: /reset_cooldown user:@username
