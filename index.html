A self-written bot for the Discord server of fucking scum, created out of boredom and curiosity >:3

For Everyone

/customize name
Changes the name of the server. Regular users are subject to a cooldown after using this command.

Parameters: text (string) - The new name for the server.
Usage: /customize name text:My Awesome Server
/customize avatar
Changes the server's avatar. It displays only the new avatar upon success. Regular users are subject to a cooldown.

Parameters: text (string, optional) - Direct URL, OR image (attachment, optional) - Uploaded file. (Must provide one).
Usage: /customize avatar text:https://... OR image:[file]
/customize channel
Renames a specific text channel. Regular users are subject to a cooldown.

Parameters: channel (TextChannel), text (string) - The new name.
Usage: /customize channel channel:#general text:new-name
/serverinfo
Displays detailed information about the server, including its name, ID, member count, creation date, owner, and the current active status of the Anarchy mode.
Usage: /serverinfo
/cooldown
Displays the current cooldown timers for a user's customization actions. If the target user is the owner, it displays a special status indicating they have no cooldowns.

Parameters: user (User, optional)
Usage: /cooldown OR /cooldown user:@username
/checkid
Fetches and displays a paginated list of all categories, text channels, and voice channels on the server, along with their corresponding IDs. Useful for finding the correct channel ID.
Usage: /checkid

For the Owner

/anarchy
Toggles "Anarchy" mode. When enabled, all cooldowns are completely bypassed for regular users. The owner is always immune to cooldowns regardless of this setting.

Parameters: status (boolean) - True to enable, False to disable.
Usage: /anarchy status:True
/clear
Purges (deletes) a specified number of recent messages in the current channel. (Note: The user executing this must also have the "Manage Messages" Discord permission).

Parameters: number (integer, 1-100)
Usage: /clear number:50
/reset_cooldown
Instantly resets all cooldown timers (name, avatar, channel) for the specified user back to zero, allowing them to use customization commands immediately.

Parameters: user (User)
Usage: /reset_cooldown user:@username
/database
Generates and sends a backup copy of the cooldown database file directly to the owner's Direct Messages.
Usage: /database
